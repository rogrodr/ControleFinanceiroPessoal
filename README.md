## 🚀 Diferenciais Técnicos e Funcionalidades

Este projeto foi desenvolvido aplicando as melhores práticas de engenharia de software:

* **Dashboards Dinâmicos:** Visualização analítica de saldos e despesas com **Chart.js**.
* **Relatórios em PDF:** Geração automática e download de relatórios detalhados das movimentações financeiras.
* **Módulos Especializados:** Gestão dedicada para **Empréstimos** e **Lembretes** de contas a pagar.
* **Autenticação Segura (JWT):** Controle de acesso via tokens, garantindo sessões seguras para os usuários.
* **Arquitetura de Dados:** Uso de **DTOs** (Data Transfer Objects) para maior segurança e senhas criptografadas no banco de dados.

## 🛠️ Stack Tecnológica

### **Backend**
* **Linguagem:** Java.
* **Framework:** Spring Boot (Security, Data JPA, Validation).
* **Banco de Dados:** PostgreSQL / H2.

### **Frontend**
* **Ferramenta de Build:** Vite.js (Performance e velocidade).
* **Biblioteca:** React.js.
* **Estilização:** CSS / Tailwind.

## 📁 Estrutura do Repositório

O repositório está organizado de forma modular para facilitar a manutenção e escalabilidade:

* **/backend**: Toda a lógica da API REST, segurança e geração de documentos.
* **/frontend**: Interface do usuário (SPA) moderna e responsiva.
* **.gitignore**: Configurado para manter o repositório limpo de arquivos de build.
